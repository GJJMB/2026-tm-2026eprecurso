Section

Layer

Foreground holds interactable Ground/Hazard tiles and entities — this is what the player collides with. Background holds purely decorative ground-equivalent sprites behind the action; it never collides and can't hold entities.

Tiles

Drag to paint. The highlighted bottom row is the ground baseline every section anchors to. Each brush below is its own sprite/color — add more in the Tileset section to paint several distinct-looking Ground/Hazard tiles in one section.

Drag to paint decorative background tiles. These sit behind everything else and never block or collide with the player.

Tileset (this section)

Each Ground/Hazard/Background brush picks a color or stretched sprite texture — "+ Add variant" makes a new brush with its own look (e.g. a second Ground sprite) that shows up as its own tool above and in the Type picker on a selected platform below. Each section keeps its own tileset, so different sections can look completely different even reused across levels.

Entities

Click an empty cell to place the selected entity (Spawn/Goal are singletons — placing a new one moves the old; Moving/enemies allow any number). Already-placed entities and platform tiles are listed in the right sidebar — select one there (or click its marker/tile here) to edit its properties.

Section JSON

Level sequence

    Drop the section files into assets/levels/sections/ and this level's file into assets/levels/levels/ (as <level id>.json). Then add "<level id>" to the sequence array in assets/levels/levels.json (if it isn't there already) to play it in-game.
